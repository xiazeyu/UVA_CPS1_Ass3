## Setup 
Install dependencies with: 
```
pip install -r requirements.txt
```

Set up your env variable to look like this: 
```
HOST = ...
PORT = ...
USER = ...
PASSWORD = ...
DATABASE = ...
```

## How to run the application
We use Flask! 
To run the website locally, use this command: 
```
python app.py
```
